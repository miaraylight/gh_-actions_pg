
# GitHub Actions Playground

This repository is a personal sandbox for experimenting with **GitHub Actions**.  
It’s where I practice building workflows, testing automation ideas, and learning how CI/CD behaves in real projects.

## 🔧 What’s Inside
- Sample workflows for pushes, pull requests, and scheduled runs  
- Experiments with linting, testing, and build pipelines  
- Tests using environment variables, caching, and matrix jobs  
- Notes on reusable workflows and best practices

## 🎯 Objectives
- Improve my understanding of GitHub Actions  
- Practice creating and debugging workflows  
- Explore secrets, artifacts, job dependencies, and more
